Formatting guidelines

SSDBM 2018 is single-blind reviewed. Therefore, authors must include their names and affiliations on the first page.

Submissions are accepted in PDF format using the new ACM proceedings LATEX or Word templates. Authors should use the sigconf proceedings template. Please see instructions at the ACM web site:

http://www.acm.org/publications/article-templates/proceedings-template.html

If you are encountering any problems using the LATEX templates, please contact
acmtexsupport@aptaracorp.com.
